# Moneyball ML Analysis

## Overview
This project analyzes Major League Baseball team performance using a Moneyball-style dataset
to understand which statistical factors are most strongly associated with team success.

## Project focus
- Data preparation and cleaning across team-level statistics
- Handling of missing values and feature selection
- Regression and machine learning models to explain team performance
- Interpretation of results and analytical insights

## Tools & technologies
- R
- Python
- Statistical modeling and machine learning
- PDF-based analytical reporting

## Notes
This project is presented as an applied analytics and modeling exercise focused on
interpretability and reasoning rather than pure prediction accuracy.
